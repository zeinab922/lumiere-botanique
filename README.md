#  Lumière Botanique – The Digital Aesthetician

##  Project Overview

Lumière Botanique is a modern e-commerce web application for skincare products.
The project simulates a real online store where users can browse products, view details, and add items to a shopping cart.

---

##  Features

*  Browse products dynamically from database
*  View product details
*  Add to cart باستخدام PHP Sessions
*  Responsive design باستخدام Tailwind CSS
*  Dynamic pages باستخدام Raw PHP

---

##  Technologies Used

* PHP (PDO)
* MySQL
* HTML5
* Tailwind CSS
* JavaScript

##  Setup Instructions

1. Install XAMPP
2. Move project to:

```
C:\xampp\htdocs\lumiere_app
```

3. Start Apache & MySQL
4. Open phpMyAdmin
5. Import:

```
database/init.sql
```

6. Run in browser:

```
http://localhost/lumiere_app
```
